# Specification: ConvResNet Implementation

## Overview

Complete the `ConvResNet` class implementation, which creates a network with residual convolutional layers followed by residual dense layers.

## Architecture Design

The `ConvResNet` should follow this structure:

1. **Convolutional layers** with residual connections and downsampling
2. **Flatten operation** to convert from convolutional to dense representation
3. **Dense residual layers** using `MLPResNet` (optional, based on parameters)
4. **Output layer** with configurable activation

## Input Parameters

The constructor already defines these parameters:

- `input_channels` (int): number of input channels
- `conv_layers_params` (dict): configuration for convolutional layers
  - `channels_mult` (list): channel multipliers (e.g., [8, 16, 32])
  - `kernels` (list): kernel sizes (e.g., [3, 3, 3])
  - `activation` (nn.Module): activation function (e.g., nn.ReLU())
  - `conv_kwargs` (dict): additional kwargs for conv layers
- `mlpResNet_params` (dict): parameters to pass to MLPResNet constructor
- `output_size` (int or None): size of output layer
- `output_layer_activation` (nn.Module or None): activation for output layer
- `output_layer_kwargs` (dict): additional kwargs for output layer
- `use_dropout` (bool or float): dropout configuration

## Implementation Requirements

### 2. Create Convolutional Layers with Residuals

Build convolutional layers using a pattern similar to UNet components:

- Use `Downsample()` layers from the same file (lines 179-221)
- Consider using residual blocks similar to `ResidualBlock` pattern from `mlp.py`
- For a simple implementation, create a `nn.Sequential()` with alternating:
  - Conv1d layers with parameters from `conv_layers_params`
  - `Downsample()` layers between conv blocks
- add activation and optional dropout to the list of sequential layers; follow the code design in the class MLPResNet()

Extract parameters:
```python
channels_mult = conv_layers_params.get('channels_mult', [8, 16, 32])
kernels = conv_layers_params.get('kernels', [3, 3, 3])
activation = conv_layers_params.get('activation', nn.ReLU())
conv_kwargs = conv_layers_params.get('conv_kwargs', {})
```

Create layers:
```python
self.conv_layers = nn.ModuleList()
in_channels = input_channels
for mult, kernel_size in zip(channels_mult, kernels):
    out_channels = mult * input_channels
    layer = Downsample(in_channels, out_channels, kernel_size,
                      activation=activation, dropout=self.dropout,
                      **conv_kwargs)
    self.conv_layers.append(layer)
    in_channels = out_channels
```

Track the output size after convolution layers for the flatten operation.

### 3. Create Dense Layers using MLPResNet

After flattening the convolutional output:

```python
if mlpResNet_params:
    # determine input size for MLPResNet after flattening
    # this needs to be calculated or passed as a parameter
    mlpResNet_input_size = ...  # calculate based on conv output
    
    self.mlp_resnet = MLPResNet(

    block_out_size = self.mlp_resnet.output_size
else:
    self.mlp_resnet = None

```

### 5. Implement Forward Method

The forward method signature is:
```python
def forward(self, x, **h_kwargs):
```

Where `h_kwargs` allows passing hidden inputs to dense layers (e.g., `h0=`, `h1=`, ...).

Implementation steps:
1. Validate input channels
2. Apply convolutional layers sequentially
3. Flatten the output: `h = torch.flatten(h, 1)`
4. If `self.mlp_resnet` exists, pass through it with `h_kwargs`
5. Apply output layer if it exists

```python
assert x.size(1) == self.input_channels, f"{x.size(1)=}, {self.input_channels=}"
h = x

# apply convolutional layers
for layer in self.conv_layers:
    h = layer(h)

# flatten for dense layers
h = torch.flatten(h, 1)

# apply dense residual network if configured
if self.mlp_resnet is not None:
    h = self.mlp_resnet(h, **h_kwargs)

# apply output layer
if self.output_layer is not None:
    y = self.output_layer(h)
else:
    y = h

return y
```

### 6. Implement Parameter Initialization

Follow the pattern from `ConvNet1D.init_parameters()` (lines 101-114):

```python
def init_parameters(self):
    """Initializes the values of trainable parameters."""
    # initialize convolutional layers
    activation = self.conv_layers_params.get('activation')
    gain = _get_gain(activation)
    for layer in self.conv_layers:
        if hasattr(layer, 'layer'):
            _set_init_parameters(layer.layer, gain)
        else:
            _set_init_parameters(layer, gain)
    
    # initialize MLPResNet (calls its own init_parameters)
    if self.mlp_resnet is not None:
        self.mlp_resnet.init_parameters()
    
    # initialize output layer
    if self.output_layer is not None:
        gain = _get_gain(self.output_layer_activation)
        if isinstance(self.output_layer, nn.Sequential):
            _set_init_parameters(self.output_layer.layer, gain, bias_scale=0.0)
        else:
            _set_init_parameters(self.output_layer, gain, bias_scale=0.0)
```

## Design Considerations

### Residual Connections

Consider two approaches:

**Option 1:** Use existing `Downsample` layers (simpler, already has activation/dropout)

**Option 2:** Create custom residual blocks with skip connections:
- Similar to `ResidualBlock` in `mlp.py` (lines 292-362)
- Each block contains: conv layer → normalization → activation → conv layer
- Skip connection from input to output
- Combine skip and block outputs: `y = 0.5*skip + 0.5*block`

### Flatten Size Calculation

The flattened size after conv layers depends on:
- Input length
- Number of downsampling operations
- Output channels of last conv layer

This may need to be:
- Passed as a parameter (`mlpResNet_input_size`)
- Calculated dynamically in `__init__` using a dummy forward pass
- Inferred from a required parameter in `mlpResNet_params`

### Hidden Layer Inputs

The `**h_kwargs` should only be used if `mlp_resnet` exists, matching the pattern in `MLPResNet` where hidden inputs are injected at specific residual blocks.

def test_ConvResNet():
    """Test ConvResNet architecture."""
    batch_size = 4
    input_channels = 1
    input_length = 64
    output_size = 10

        input_channels=input_channels,

    h0 = torch.randn(batch_size, 8)
    y = net(x, h0=h0)
    assert y.shape == (batch_size, output_size)
    
    print("ConvResNet tests passed")

if __name__ == '__main__':
    test_ConvResNet()
